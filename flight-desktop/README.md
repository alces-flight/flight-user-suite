# Flight Desktop

Flight Desktop allows users to create and manage virtual desktop sessions of
various environment types.

## Usage

* Check system dependencies are satisfied
  ```sh
  flight desktop doctor
  ```
* List available desktop types
  ```sh
  flight desktop avail
  ```
* Start a new session
  ```sh
  flight desktop start <desktop type>
  ```
* List sessions
  ```sh
  flight desktop list
  ```
* Show details for a session
  ```sh
  flight desktop show <session name>
  ```
* Kill a session
  ```sh
  flight desktop kill <session name>
  ```
